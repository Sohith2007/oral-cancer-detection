# Conventions

Coding standards and patterns followed in the codebase.

## JavaScript / React
- **Syntax**: Modern ESM (import/export).
- **Components**: Functional components with hooks.
- **State Management**: React `useState` and `useEffect` (currently local to pages).
- **Styling**: Tailwind CSS classes directly in JSX.
- **Formatting**: Managed by ESLint (Vite defaults).

## Backend (Planned)
- **Framework**: FastAPI (Python).
- **Standard**: RESTful API design.
- **Folder Structure**: Models, Schemas, Services, and Uploads (as per user plan).
