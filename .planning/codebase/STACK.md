# Tech Stack

Overview of the technology stack used in the Oral Cancer Multimodal AI System.

## Frontend
- **Framework**: React 19.2.5 (Vite 8.0.10)
- **Routing**: react-router-dom 7.14.2
- **Styling**: Tailwind CSS 4.2.4
- **Icons**: Lucide React 1.9.0
- **Visualization**: Recharts 3.8.1
- **Language**: JavaScript (ESM)
- **Linting**: ESLint 10.2.1

## Backend
- **Planned**: FastAPI (Python 3.10+)
- **Status**: Not yet implemented

## Tooling
- **Build Tool**: Vite
- **Package Manager**: npm
